# Installation

## Stable release

To install Teaching Automation Toolkit, run this command in your terminal:

```sh
uv add teacher-automation
```

Or if you prefer to use `pip`:

```sh
pip install teacher-automation
```

## From source

The source files for Teaching Automation Toolkit can be downloaded from the [Github repo](https://github.com/maescobar132/teacher_automation).

You can either clone the public repository:

```sh
git clone git://github.com/maescobar132/teacher_automation
```

Or download the [tarball](https://github.com/maescobar132/teacher_automation/tarball/master):

```sh
curl -OJL https://github.com/maescobar132/teacher_automation/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd teacher_automation
uv pip install .
```
